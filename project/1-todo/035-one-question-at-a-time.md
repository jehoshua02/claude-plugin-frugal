# Ask One Question at a Time

## Abstract

Add a rule: when clarification is needed, ask questions one at a time instead of dumping a page of questions. Reduces cognitive load and keeps conversation focused.

## Priority: 035

- Value: 7/10 — Saves tokens on both sides, reduces conversation complexity
- Momentum: 1/10 — Fresh idea
- Effort: 2/10 — One-line rule addition
- Risk: 1/10 — No downside

## Timeline

- Refined: 2026-04-28

## Details

Fewer questions per turn means shorter responses, simpler user replies, and fewer parallel threads. All of which reduce token usage.
